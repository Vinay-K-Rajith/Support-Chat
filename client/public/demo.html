How to Embed

To embed this widget on your website, simply add this to your HTML:

<div id="school-chat-widget"></div> <script src="https://your-ngrok-url.ngrok.io/assets/embed.js"></script>

Note: Replace your-ngrok-url.ngrok.io with your actual ngrok URL.

JavaScript API

You can also control the widget programmatically:

// Open the chat widget SchoolChatWidget.open(); // Close the chat widget SchoolChatWidget.close();
